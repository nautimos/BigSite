❓ Perguntas Frequentes

Como obter uma API key do Google Gemini?

Acesse Google AI Studio, faça login e crie uma nova API key. É gratuito com limites de uso.

A API Key é gratuita?

Sim! Google oferece quota gratuita generosa (15 requisições/minuto). Para uso mais intenso, pode ser necessário billing.

Por que o botão "Verificar Script no Site" não funciona?

Primeiro você precisa gerar e instalar o script de conexão no seu site. Vá na aba CONECTOR e clique em "Gerar Script de Instalação".

Como funciona a análise SEO?

O Big Site captura o HTML da página, envia para o Google Gemini 2.5 Pro que analisa e retorna sugestões personalizadas de otimização.

Meus dados são seguros?

Sim! Todos os dados ficam armazenados LOCALMENTE no seu navegador. Não enviamos nada para servidores externos (exceto para o Google Gemini durante análises).

Qual modelo Gemini é usado?

Usamos gemini-2.5-pro (o mais avançado) com fallback automático para gemini-2.5-flash se necessário.

🐛 Problemas Comuns

Erro: "API key inválida"

✅ Verifique se copiou a chave completa
✅ Chave deve começar com "AIza"
✅ Crie uma nova key em Google AI Studio

Erro: "Nenhum modelo disponível"

✅ Ative a "Generative Language API" no Google Cloud
✅ Aguarde alguns minutos após criar a API key
✅ Verifique se tem quota disponível

Backend não conecta

✅ Certifique-se que o servidor está rodando (npm run dev)
✅ Verifique se a URL está correta (http://127.0.0.1:3000)
✅ Confira o console do servidor para erros
